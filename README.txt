PT-BR:
Título: Script DirBuster
Descrição: Script DisBuster para descoberta de diretórios e arquivos presentes em um site ou servidor alvo, desenvolvido em python 3.6.2.
Para a instalação, execute o comando "pip install -r requirements.txt" no terminal. Você pode alterar a lista de palavras, mas mantê-la no mesmo diretório e executar o arquivo python.
Para utilizar o DirBuster deve ser usado o comando "python dirbust.py url wordlist". Por exemplo:
python dirbust.py http://example.com/ wordlist.txt /

EN-US:
Title: Script DirBuster
Description: DisBuster script for discovering directories and files present on a target website or server, developed in python 3.6.2.
For installation, run the command "pip install -r requirements.txt" in terminal. You can change the wordlist but keep it in the same directory and run the python file.
To use DirBuster, use the command "python dirbust.py url wordlist". For example:
python dirbust.py http://example.com/ wordlist.txt /
